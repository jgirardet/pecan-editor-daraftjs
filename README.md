# PECAN EDITOR

A `draft-js` editor used in [MyCartable](<[MyCartble](https://github.com/jgirardet/mycartable)>).

# Using

```js
import { PecanEditor } from "./components/PecanEditor";

const MyEditor = () => {
  return <PecanEditor />;
};
```
